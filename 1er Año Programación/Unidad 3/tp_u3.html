Tomando Decisiones (IF / ELSE)

Introducción a la Programación  ·  1er Año

Unidad 3 Trabajo Práctico

Objetivo de hoy: Programar a la compu para que tome decisiones usando if (si pasa esto...) y else (sino, hace esto otro).

Misión 1: ¿Ya podés votar?

El Sistema Electoral

En Argentina se puede votar a partir de los 16 años. Vamos a programar una máquina que pregunte tu edad y te diga si ya podés votar o si tenés que esperar un poco.

  • Paso 1: En un bloque de Colab, copien este código:
    edad = input("¿Cuántos años tenés? ")
    edad = int(edad) # Esto convierte el texto en NÚMERO (Integer)

    if edad >= 16:
        print("¡Ya podés ir a votar!")
    else:
        print("Todavía te falta crecer un poco.")
  • Atención: Noten que la palabra print está "metida hacia adentro" (sangría o tabulación). ¡Python necesita ese espacio para saber qué código va adentro del IF y del ELSE!
  • Pruébenlo: Denle Play y prueben poniendo primero el número 12 y luego el 18.

Misión 2: El Portero Virtual (Ciberseguridad)

Ahora vamos a armar el sistema de seguridad de una base secreta militar. Como la contraseña es texto (String), ¡no hace falta usar int()!

  • Desafío: Creen un programa nuevo que le pida una contraseña al usuario.
  • Regla 1: Si el usuario escribe "Innova2024" (nuestra clave secreta), la compu tiene que decir: "Acceso Permitido. Bienvenido Agente."
  • Regla 2: Si escribe cualquier otra cosa, debe usar el else para decir: "¡INTRUSO DETECTADO! Llamando a la policía."
💡

Pista: Para comparar si dos palabras son EXACTAMENTE iguales, en programación no se usa un solo igual (=), ¡se usa el doble igual: ==!
