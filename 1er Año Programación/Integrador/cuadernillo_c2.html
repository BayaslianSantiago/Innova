Apuntes Clave de Programación  ·  Ciclos & Listas

🔁 Ciclo FOR (La máquina de repetir)

Le enseñamos a la compu a repetir cosas. El range(veces) indica cuántas veces. ¡No te olvides de los : y la sangría!

for i in range(5):
    print("Esto se imprime 5 veces")

🎒 Listas (Tu Inventario)

Sirven para guardar MUCHAS cosas juntas usando corchetes [ ] y separando por comas.

mochila = ["Espada", "Poción", "Llave"]
print(mochila)

🤯 El Truco del Cero

Las computadoras son raras y empiezan a contar desde el número CERO. El primer objeto de tu mochila es el número 0.

print(mochila[0]) # Imprime Espada
print(mochila[1]) # Imprime Poción

✨ Guardar nuevos tesoros (.append)

Para agregar algo nuevo al final de la mochila sin borrar lo demás, usamos .append().

tesoro = input("¿Qué agarraste? ")
mochila.append(tesoro)
print(mochila)

🔍 El Escáner Mágico (FOR en LISTAS)

¡El combo final! Usamos el ciclo for para leer todo el inventario, y un if escondido adentro (con doble sangría) para buscar algo en particular.

for objeto in mochila:
    if objeto == "Oro":
        print("¡Encontré oro!")

Ojo: Si olvidás la sangría, el if se escapa del ciclo.
