💬 Hablar y Escuchar

Para hablar (mostrar en pantalla): Usamos print.

print("Hola, soy la compu")

Para escuchar (hacer preguntas): Usamos input.

input("¿Cómo te llamás? ")

📦 Variables (Cajas Mágicas)

Las usamos para guardar lo que responde el usuario y no olvidarlo.

nombre = input("Dime tu nombre: ")
print("Hola", nombre)

🔤 Tipos de Cajas (Tipos de Datos)

Texto (String): Va siempre con comillas.
Ejemplo: "Hola", "Mago", "123"

Números (Integer): Sin comillas, para hacer cuentas.
Ejemplo: 15, 100

¡Truco mágico! Para convertir un texto en un número de verdad, usamos int():

edad = input("Edad: ")
edad = int(edad) # Ahora sí es un número

🔀 Decisiones (IF / ELIF / ELSE)

Le enseñamos a la compu a elegir caminos. elif agrega más opciones intermedias. ¡Ojo a la sangría (los espacios adentro)!

if clase == "Mago":
    print("Magia")
elif clase == "Orco":
    print("Fuerza")
else:
    print("Error: No existe")

⚖️ Operadores de Comparación

Los usamos junto con el if para comparar cosas:

  • == Exactamente igual (¡DOBLE IGUAL!)
  • != Diferente a
  • > Mayor que (Ej: 18 > 12)
  • < Menor que (Ej: 10 < 15)
  • >= Mayor o igual que
  • <= Menor o igual que

🛡️ Contraseña Perfecta

Recordá las 3 reglas de oro para la Ciberseguridad:

  • Larga: Más de 8 letras.
  • Rara: Mezclar MAYÚSCULAS, números y símbolos (como @, !).
  • Sin datos personales: No usar nombres ni fechas.
