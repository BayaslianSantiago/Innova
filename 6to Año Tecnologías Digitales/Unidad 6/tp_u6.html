Hello Streamlit (Setup Web)

Ciencia de Datos y Análisis  ·  6to Año (2do Cuatrimestre)

Unidad 6 Trabajo Práctico

El Contexto: Antes de construir el mega proyecto final del cuatrimestre, debemos comprobar que los servidores de GitHub pueden levantar una página web en vivo. Crearán una aplicación web de 3 líneas de código.

Misión 1: Escribir la Web

  • Entorno: Abran su repositorio innova-data-lab en Codespaces.
  • El Archivo: Creen un archivo nuevo llamado app.py. Copien y peguen EXACTAMENTE este código:
    import streamlit as st st.title("Mi primera Web de Datos") st.write("¡Los servidores están en línea!")

Misión 2: Lanzar el Servidor

En el trabajo práctico anterior, usábamos el botón de "Play" para ejecutar cosas en la terminal. Ahora tenemos que decirle a la computadora que actúe como un Servidor Web que mantenga la página abierta.

  • Abran la Terminal en la parte inferior de la pantalla de Codespaces.
  • Escriban el siguiente comando y presionen Enter:
    streamlit run app.py
  • El Milagro: GitHub detectará el servidor y les abrirá automáticamente una nueva pestaña en el navegador con su página web funcionando.

Misión 3: El Diseño Interactivo (Preview)

✨

Magia Instantánea: Mantengan la pestaña de su nueva web abierta, vuelvan al código, agreguen debajo un st.balloons(), guarden el archivo y vean qué pasa en la web.

Formato de Entrega

Acuérdense de hacer el Stage, Commit y Push de su archivo app.py. Entreguen en el Classroom una captura de pantalla de la página web funcionando con los globos.
