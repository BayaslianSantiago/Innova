Unidad 5 Trabajo Práctico

El Contexto: El Director les envió un archivo de ventas que fue corrompido por un error del sistema (tiene filas en blanco). Deben usar Pandas en GitHub Codespaces para leerlo, borrar la basura, y decirle al Director cuánto dinero exacto hizo cada Sucursal.

Misión 1: Preparar la Basura (Setup)

  • Entorno: Abran su repositorio innova-data-lab en GitHub y lancen su entorno Codespaces.
  • Dataset Corrupto: Creen un archivo nuevo llamado ventas_sucias.csv. Copien y peguen EXACTAMENTE el siguiente texto adentro y guarden:
    Sucursal,Producto,Venta Mendoza,Zapatillas,1500 Cordoba,Remeras,800 BuenosAires,Pantalones, Mendoza,Remeras,700 Cordoba,,500 BuenosAires,Zapatillas,2000

Misión 2: El Script de Limpieza

Creen un archivo llamado analisis.py. Escriban un script de Python que haga lo siguiente (usen las diapositivas como guía):

  • Importe la librería pandas.
  • Lea el archivo ventas_sucias.csv usando pd.read_csv() y lo guarde en una variable llamada df.
  • Aplique el método .dropna() sobre ese DataFrame para eliminar las filas que no tienen datos (la venta de BuenosAires y el producto de Cordoba).
  • Utilice .groupby('Sucursal').sum() sobre el DataFrame ya limpio para sumar todo el dinero, y finalmente use print() para mostrarlo por pantalla.
  • Ejecución: Corran el archivo en la terminal (usando el botón Play arriba a la derecha). Verifiquen que la consola no imprima errores y muestre la suma.

Misión 3: El Analista vs la IA
