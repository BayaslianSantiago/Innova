☁️ U4: Control de Versiones

El flujo de guardado en la nube:

  • Stage (+): Preparar los archivos que queremos subir.
  • Commit: Sacar una foto permanente al código. Siempre escribir un mensaje descriptivo.
  • Push (Sync): Enviar los cambios desde Codespaces hacia la página de GitHub.

Regla de Oro: En tecnología NUNCA pasamos código por mail ni pendrive.

🐼 U5: Manipulación (Pandas)

Comandos Esenciales de Python:

  • df = pd.read_csv('archivo.csv')
    Carga una tabla en memoria.
  • df = df.dropna()
    Limpia la base de datos (borra filas con casilleros en blanco/nulos).
  • df.groupby('Columna').sum()
    Agrupa categorías iguales y suma los totales (estilo tabla dinámica).
🎈 U6: Dashboard Web (Streamlit)

Levantar el Servidor:

Para ver la página web que programamos, hay que escribir en la Terminal:
streamlit run app.py

Componentes de la Capa 1 (Resumen):

  • c1, c2 = st.columns(2) (Crea columnas invisibles).
  • c1.metric("Ventas", "100") (Dibuja el KPI).

Componentes Visuales (Capas 2 y 3):

  • st.title("Texto") (Título grande).
  • st.bar_chart(datos) (Dibuja gráfico de barras).
  • st.line_chart(datos) (Dibuja gráfico de líneas temporales).
  • st.dataframe(df) (Muestra la tabla de Excel interactiva completa).
  • st.balloons() (Magia de celebración).
