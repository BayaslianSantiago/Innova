Cultura Tech

La Tragedia del Archivo ZIP

El final del `dashboard_final_v5_ahorasi.zip`

En el colegio y en la universidad, la gente trabaja en equipo mandándose archivos de Word por mail o por WhatsApp. Cuando programás un sistema que vale millones de dólares, si dos personas editan el mismo archivo y lo sobreescriben, la empresa colapsa.

El Problema Local

Si tu computadora se rompe, el código se pierde. Si otro programador tiene Windows y vos Mac, el código falla ("En mi máquina sí funcionaba").

La Solución: Git y GitHub

Un sistema global de Control de Versiones. Cada línea de código queda registrada: quién la escribió, a qué hora y por qué. Y todo vive en la Nube.

Conceptos Clave

El Ciclo de Vida del Código

Guardar archivos en tecnología tiene 3 pasos.

1. STAGE (Preparar)

Le avisás al sistema qué archivos querés guardar en el próximo paquete.

git add .

2. COMMIT (Foto)

Le sacás una "foto" permanente al código con un mensaje explicando qué hiciste.

git commit -m "Agrego KPI"

3. PUSH (Subir)

Sincronizás esa foto desde tu computadora hacia la nube (GitHub) para que el equipo la vea.

git push

Nota: Nosotros usaremos los botones de la interfaz gráfica de VS Code para hacer esto, ¡pero por debajo la computadora ejecuta exactamente esos comandos!

Tu Máquina Virtual Gratuita

A partir de hoy, no instalaremos Python en nuestras computadoras.


Usaremos GitHub Codespaces. Es una computadora completa ubicada en un servidor de Microsoft en Estados Unidos, a la que accedemos directamente desde nuestro Google Chrome.

Ventaja Absoluta: Si tu computadora de la escuela es lenta, no importa. El código se está ejecutando en los servidores súper rápidos de la nube. Solamente necesitamos internet.
