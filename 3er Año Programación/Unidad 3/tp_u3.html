Bases de Datos (Diccionarios)

Programación y Desarrollo Web  ·  3er Año (1er Cuatrimestre)

Unidad 3 Trabajo Práctico

Objetivo de hoy: Organizar información de manera profesional utilizando Diccionarios en Python, empleando la estructura de Clave-Valor para simular una base de datos de la Selección Argentina.

Misión 1: La Ficha Técnica

La Scaloneta

Vamos a construir el sistema que maneja las estadísticas de nuestros campeones. ¡No podemos usar listas porque nadie se acordaría en qué posición está cada dato!

  • Creación: En un nuevo archivo de Colab, creen un diccionario llamado jugador.
  • Datos: Elijan a su jugador favorito de la Selección Argentina y agréguenle las siguientes claves: "nombre", "edad", "dorsal" y "posicion".
  • Sintaxis: Recuerden usar llaves {} para abrir y cerrar el diccionario, dos puntos : para separar clave y valor, y comas para separar cada fila.

Misión 2: El Locutor del Estadio

Ahora que tenemos la información ordenada, necesitamos extraerla para anunciarla por los altoparlantes del estadio.

  • Extracción: Usen print() para mostrar una oración concatenando texto normal y datos extraídos del diccionario.
  • El Anuncio: Debe decir algo como: "¡Ingresa al campo el jugador [su_nombre], con la camiseta número [su_dorsal]!"
🤖

Uso Obligatorio de IA: Es muy común olvidarse una coma o cerrar mal las comillas al escribir diccionarios. Si Colab les da un error (SyntaxError o KeyError), deben ir a ChatGPT, usar la fórmula del prompt que aprendimos y pedirle que les encuentre el error de sintaxis.

Misión 3: El Mercado de Pases

La información no es estática, cambia todo el tiempo. ¡Acaba de ocurrir un evento importante!

  • Actualización: Escriban código abajo para modificar la "edad" del jugador (sumándole 1 año porque cumplió años).
  • Nuevo Dato: Agreguen una nueva clave llamada "club_actual" y asígnenle el nombre del equipo en el que juega actualmente.
  • Verificación: Hagan un `print(jugador)` al final de todo para mostrar que la ficha técnica se actualizó correctamente en la base de datos.
