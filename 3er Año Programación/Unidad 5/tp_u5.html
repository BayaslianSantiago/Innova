Interceptando Satélites (APIs)

Programación y Desarrollo Web  ·  3er Año (2do Cuatrimestre)

Unidad 5 Trabajo Práctico

Objetivo de hoy: Conectarse de forma segura (HTTPS) a un servidor meteorológico mundial, descargar los datos crudos (JSON) y extraer únicamente la temperatura actual utilizando Diccionarios.

Misión 1: La Conexión Segura

Vamos a usar la librería requests que trae Python para simular ser un navegador y pedirle la temperatura a la API gratuita de Open-Meteo.

  • Preparación: Escriban import requests en la primera celda.
  • El Objetivo: Guarden este enlace exacto en una variable llamada url:
    "https://api.open-meteo.com/v1/forecast?latitude=-34.61&longitude=-58.37¤t_weather=true" (¡Asegúrense de que empiece con https!).
  • El Pedido GET: Hagan la solicitud usando: respuesta = requests.get(url)

Misión 2: Decodificando el JSON

La respuesta que nos da el servidor es un paquete gigante de texto incomprensible. Tenemos que convertirlo al idioma que Python entiende: Los Diccionarios.

  • Conversión: Escriban datos = respuesta.json() para transformar ese texto en un Diccionario gigante.
  • Visualización: Hagan un print(datos) para ver toda la matriz de información meteorológica que acabamos de descargar.

Misión 3: Extracción Quirúrgica

Toda esa información junta no nos sirve. El usuario solo quiere saber qué temperatura hace ahora mismo.

  • El Diccionario Anidado: Si miran bien el JSON, adentro del diccionario gigante hay una clave llamada "current_weather", y adentro de ella hay otra clave llamada "temperature".
  • La Extracción: Creen una variable llamada clima_actual y guarden ahí el dato navegando entre las llaves.
    (Tip: Se usan los corchetes pegados uno al lado del otro, ej: datos["diccionario1"]["diccionario2"]).
  • El Mensaje Final: Hagan un print amigable usando concatenación: "Atención: La temperatura actual es de [numero] grados."
