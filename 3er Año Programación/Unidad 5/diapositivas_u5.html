Concepto

¿Qué es una API?

El Mesero Cibernético.

API significa Interfaz de Programación de Aplicaciones. En palabras simples, es un mensajero.

Imaginá un restaurante: Vos sos el Código, la cocina es el Servidor. No podés entrar a la cocina a agarrar los ingredientes. Necesitás un Mesero (La API) que lleve tu pedido y te traiga la comida lista.

Extracción

El Idioma JSON

El secreto revelado.

Cuando el mesero (API) te trae los datos de internet, te los entrega en un formato universal llamado JSON. Suena complejo, pero miren bien el código de abajo...

# ¿A qué se parece el JSON?
{
  "ciudad": "Buenos Aires",
  "clima": {
    "temperatura": "22°C"
  }
}

¡Exacto! El JSON es exactamente igual a los Diccionarios que aprendimos en la Unidad 3.

Ciberseguridad

HTTP vs HTTPS

El Peligro del Wi-Fi Público.

Para pedir datos a la API usamos un método llamado GET (Obtener). Este pedido viaja por el aire en milisegundos.

HTTP (Inseguro): Si tu pedido viaja por HTTP en el Wi-Fi de un café, cualquier hacker en esa misma red puede leer exactamente qué estás pidiendo (Man-in-the-Middle).

HTTPS (Seguro): La letra 'S' significa *Secure*. Encripta tu mensaje. Si el hacker lo intercepta, solo verá basura ilegible.
