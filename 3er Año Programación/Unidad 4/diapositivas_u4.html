La Regla de Oro

Principio D.R.Y.

Don't Repeat Yourself (No te repitas a ti mismo).

Copiar y pegar el mismo código 50 veces es el error más común de un novato. Si tenés que cambiar algo, vas a tener que corregirlo 50 veces.

La solución: Envolver ese código en una "caja" y ponerle un nombre. Así, solo lo escribís 1 vez y lo llamás cuando lo necesites.

Fabricando

Las Funciones (`def`)

Creando tus propias herramientas.

Una función es una pequeña fábrica: le entran materias primas (Parámetros) y sale un producto terminado (`return`). Usamos la palabra def para definirla.

# 1. Definimos la herramienta
def escanear_amenaza(nivel):
  if nivel > 50:
    return "¡Peligro!"
  else:
    return "Sistema Seguro"

# 2. La usamos cuando queremos
print(escaner_amenaza(80)) # Imprime: ¡Peligro!
El Arsenal Externo

Librerías (`import`)

No reinventes la rueda.

Si necesitás una herramienta matemática compleja, generar números aleatorios, o saber la hora mundial, alguien ya lo programó por vos.

# Descargamos el módulo "random" (aleatorio)
import random

# Lo usamos para generar un número del 1 al 10
dado = random.randint(1, 10)
print("El dado tiró: " + str(dado))
