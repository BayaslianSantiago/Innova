🛠️ U4: Funciones y Librerías

Crear nuestra propia herramienta para no repetir código:

def sumar(a, b):
  resultado = a + b
  return resultado

print(sumar(10, 5)) # Imprime 15

🌐 U5: Conexión Satelital (APIs)

Hacer una petición GET a un servidor seguro (HTTPS):

import requests
url = "https://api.clima.com"
respuesta = requests.get(url)

Extraer de un diccionario JSON anidado:

datos = respuesta.json()
# Para sacar la temperatura:
temp = datos["clima_actual"]["temperatura"]
🔐 U6: Strings y Criptografía

Limpiar la entrada del teclado:

texto = " HOLA MUNDO "
texto = texto.strip() # Borra espacios
texto = texto.lower() # Pasa a minúscula

Criptografía Clásica: El Código César esconde el texto desplazando las letras del alfabeto hacia adelante (+3). A se convierte en D.
