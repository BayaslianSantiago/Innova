🐍 2. Repaso de Python (1er Año)

Variables: Guardan datos simples en la memoria.

edad = 15 nombre = "Neo"

Condicionales (IF): Toman decisiones basadas en lógica.

if edad >= 18: print("Acceso concedido")

Ciclos (FOR): Repiten acciones, ideal para recorrer listas.

for jugador in equipo: print(jugador)

🤖 3. El Copiloto IA (Prompts)

La IA (ChatGPT, Copilot, Gemini) funciona mejor si le das instrucciones precisas llamadas Prompts.

La Fórmula Mágica:

  • Contexto: "Actúa como un experto en Python."
  • Tarea: "Encuentra el error en este código."
  • Formato: "Explícalo en un solo párrafo corto."

Alucinaciones: A veces la IA inventa respuestas que parecen reales. NUNCA copies y pegues código sin leerlo y probarlo.

📦 4. Listas vs Diccionarios

Listas []: Guardan datos por su POSICIÓN. No tienen "nombre".

# El 10 es la edad, pero la máquina no lo sabe datos = ["Leo", 10] print(datos[0]) # Imprime "Leo"

Diccionarios {}: Guardan datos por CLAVE-VALOR. Cada dato tiene una etiqueta.

# El dato tiene significado jugador = { "nombre": "Messi", "dorsal": 10 }

🛠️ 5. Operaciones con Diccionarios

  • Leer un dato: jugador["nombre"]
  • Modificar: jugador["dorsal"] = 30
  • Agregar nuevo: jugador["club"] = "Miami"

Se pueden meter diccionarios adentro de una lista para armar una base de datos gigante:

plantilla = [ {"nombre": "Messi", "n": 10}, {"nombre": "Dibu", "n": 23} ]
