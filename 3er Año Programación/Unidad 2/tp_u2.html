Misión 1: El Portfolio Híbrido

Colab, viejo amigo

¿Se acuerdan cómo funcionaba Colab en 1er Año? Podemos mezclar texto normal (Markdown) con celdas de código. Vamos a armar nuestra presentación.

  • Markdown: Creen una celda de Texto en Colab. Usen el símbolo # para poner un título con su nombre, y abajo escriban algo sobre ustedes.
  • Python: Creen una celda de Código debajo, y usen la función print() para que la compu salude al mundo.

Misión 2: El Código Roto (Ingeniería de Prompts)

Un programador inexperto dejó este código Python tirado. Intenta contar del 0 al 4 y luego avisar que terminó, pero ¡está roto y tira error!

for i in range(5)
print("Contando: " + i)
print("Fin del conteo")
  • La Copia: Copien y peguen el código roto en una nueva celda de Colab. Si le dan Play, verán que tira SyntaxError.
  • El Copiloto: Abran ChatGPT, Gemini o Claude en otra pestaña.
  • El Prompt Mágico: Utilizando la fórmula vista en clase (Contexto + Tarea + Formato), pídanle a la IA que repare el código.
    (Ejemplo: "Actúa como un profesor de Python. Encuentra los errores de este código y devuélvemelo corregido. Explicámelo en 2 oraciones.")

Misión 3: El Reporte de la Operación

🚨

¡Atención! La regla de oro del desarrollador es nunca confiar a ciegas. Tienen que probar el código reparado por la IA en Colab para verificar que funcione (que no haya "alucinado").

  • Evidencia: Creen una celda de Texto en su Colab y peguen literalmente cuál fue el Prompt exacto que le mandaron a la IA.
  • Solución: Abajo, en una celda de código, peguen el código ya reparado y funcionando.
