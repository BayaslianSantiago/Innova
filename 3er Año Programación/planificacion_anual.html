Planificación Anual: Programación

Profesor Bayaslian Santiago

1. Fundamentación

Esta materia arrancará con un enfoque visual (HTML) para enganchar su interés, y luego explotará el entorno Google Colaboratory (Colab) con Python.

Al igual que en 1er año, la currícula mantendrá un enfoque transversal sobre tres pilares: los Datos (cómo consumirlos y estructurarlos), la Ciberseguridad (cómo encriptarlos y transmitirlos de forma segura), y la Inteligencia Artificial (uso ético de copilotos de código para acelerar el desarrollo). Aprenderán a crear programas modulares que se comuniquen con el mundo exterior directamente desde la nube.

2. Objetivos de Aprendizaje

  • Comprender la estructura de la Web (HTML) y su relación con la ciberseguridad.
  • Manejar estructuras de datos complejas (Diccionarios) y entender el formato JSON.
  • Modularizar el código utilizando Funciones (def) e incorporar Librerías externas (import).
  • Conectar programas con servicios web mediante APIs públicas.
  • Usar responsablemente herramientas de IA Generativa para asistir en la escritura y depuración de código.

3. Ejes Temáticos (Formato Cuatrimestral)

Primer Cuatrimestre: "De la Web a los Datos"

  • Unidad 1: Magia Visual (HTML). Etiquetas básicas. Ciberseguridad 101: por qué no guardar información sensible ni contraseñas en el código fuente (HTML estático).
    • Evaluación: TP Unidad 1 (Crear una página web estática sencilla).
  • Unidad 2: Colab y Repaso Asistido por IA. Repaso de Python. Introducción a los asistentes de código (ej. ChatGPT/Copilot): cómo hacerles buenas preguntas (Prompt Engineering básico) para encontrar errores.
    • Evaluación: TP Unidad 2 (Crear un mini-portfolio personal en Colab combinando Markdown y código).
  • Unidad 3: Diccionarios. La estructura Clave-Valor. Cómo se agrupan los datos masivos en la industria de la tecnología.
    • Evaluación: TP Unidad 3 (Crear un diccionario de "Ficha de Personaje" o "Pokedex").
  • Proyecto Integrador C1: Sistema de Gestión interactivo en Colab utilizando Diccionarios y documentado con IA.

Segundo Cuatrimestre: "Conectados al Mundo (APIs)"

  • Unidad 4: Modularización y Librerías. Crear herramientas propias (def) y reutilizar herramientas de otros mediante Módulos (import). Qué es una librería externa.
    • Evaluación: TP Unidad 4 (Importar una librería estándar como math o datetime y crear una función que la utilice).
  • Unidad 5: Introducción a APIs y JSON. Solicitudes GET con la librería requests. Riesgos básicos de interceptación de datos (HTTP vs HTTPS).
    • Evaluación: TP Unidad 5 (Extraer el clima actual de una API).
  • Unidad 6: Procesamiento de Strings y Seguridad. Limpiar datos recibidos por internet. Introducción a la criptografía.
    • Evaluación: TP Unidad 6 (Desarrollar un encriptador básico tipo Código César para enviar "mensajes secretos").
  • Proyecto Integrador C2: Un "Asistente Virtual" en texto. El alumno debe pedir un dato, conectarse a una API pública (ej: API de feriados, clima o películas), extraer la información clave del JSON usando diccionarios, y mostrarla. Deberán apoyarse en IA para resolver un problema lógico durante el desarrollo.

4. Criterios de Evaluación

El año lectivo será evaluado íntegramente mediante entregas prácticas, exposiciones y trabajo en el aula:

  • Entregas de Unidad: 1 Trabajo Práctico corto por cada una de las 6 unidades.
  • Proyectos Integradores y Defensa Oral: 1 Proyecto integrador en grupo por cuatrimestre. La evaluación incluye obligatoriamente una instancia de exposición oral frente a la clase, donde el grupo debe explicar cómo funciona su código, demostrar que lo comprenden y responder preguntas técnicas.
  • Participación Activa: Se evaluará de forma continua la predisposición, el compañerismo y la participación activa durante las clases (aportar ideas, debatir lógicas y ayudar a otros grupos a encontrar errores en sus códigos).
  • Uso Ético y Crítico de la IA: Se fomenta y espera el uso de herramientas de Inteligencia Artificial como asistentes de programación. Sin embargo, su uso debe estar acompañado de pensamiento crítico: el estudiante debe comprender, explicar y justificar cada concepto implementado por la IA, demostrando una apropiación real del contenido de la materia.
