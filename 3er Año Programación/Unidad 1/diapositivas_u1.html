El Esqueleto

¿Qué es el HTML?

El código que construye internet.

HTML significa Lenguaje de Marcado de Hipertexto. No es un lenguaje de programación, es un lenguaje de marcado. Sirve para decirle al navegador (Chrome, Safari) dónde va un título, dónde va un texto y dónde va una imagen.

<h1>Hola Mundo</h1>
<p>Este es mi primer párrafo.</p>
💡

Casi todo en HTML funciona con etiquetas de apertura <etiqueta> y etiquetas de cierre </etiqueta>.

Plataformas de Desarrollo

Para escribir código web no hace falta instalar nada. Vamos a utilizar CodePen, un editor online gratuito que usan los desarrolladores profesionales para probar ideas rápidamente.

🖥️

En CodePen tenemos tres paneles: HTML (la estructura), CSS (el diseño visual) y JS (la inteligencia interactiva). ¡Hoy nos enfocamos 100% en el HTML!

Ciberseguridad 101

El Código NO es secreto

Inspeccionando Elementos.

El código HTML se ejecuta en el navegador del usuario (es Client-Side). Esto significa que cualquier persona puede ver el código fuente de una página web haciendo Click Derecho -> Inspeccionar.

🚨

Regla de Oro de Seguridad: NUNCA, jamás, guardes contraseñas, tokens bancarios o datos privados directamente en el código HTML de tu página. ¡Cualquiera podría robarlos!

<!-- NUNCA HAGAS ESTO -->
<p>Bienvenido. La contraseña secreta del servidor es Admin1234</p>
