Unidad 1 Trabajo Práctico

Objetivo de hoy: Construir nuestra primera estructura web usando HTML en CodePen, y comprender por qué inspeccionar el código es una técnica básica de Ciberseguridad.

Misión 1: Mi Primera Web

El Arquitecto

Para construir una casa, primero necesitamos los ladrillos. En el mundo del desarrollo web, los ladrillos son las etiquetas HTML.

  • Plataforma: Ingresen a codepen.io/pen/. Vamos a trabajar únicamente en el panel que dice HTML.
  • El Título: Creen un título grande usando la etiqueta de encabezado principal: <h1>Mi nombre es [Tu Nombre]</h1>.
  • El Párrafo: Debajo del título, agreguen un párrafo explicando cuál es su tecnología, red social o videojuego favorito: <p>Me encanta...</p>.

Misión 2: Enlazando la Red

Una página web aburrida no lleva a ningún lado. Vamos a hacer honor al nombre "Hipertexto" conectando nuestra web con otra.

  • El Vínculo: Usen la etiqueta de enlace (ancla) para crear un texto clickeable.
  • El Código: Escriban debajo del párrafo:
    <a href="https://instagram.com">Ir a Instagram</a>
  • Prueba: Clickeen en el resultado (abajo) para verificar que el enlace funciona y los lleva a la página indicada.

Misión 3: El Analista de Seguridad (Hack)

Como vimos en clase, todo el código HTML de cualquier sitio se puede inspeccionar. Es una vulnerabilidad grave si el programador es descuidado.

🕵️

El profesor les va a enviar el archivo boveda_secreta.html. Es una pantalla de Login, pero el programador dejó el usuario y la contraseña escondidos directamente en el código.

  • Inspección: Abran la Bóveda Secreta, hagan Click Derecho y seleccionen "Inspeccionar" o "Ver código fuente de la página".
  • La Búsqueda: Busquen en el panel que se abre a la derecha (el DOM) un comentario en verde o analicen la parte de <script> para encontrar la contraseña (y el usuario).
  • Reporte: En su CodePen, agreguen un nuevo párrafo al final que diga: <p>La contraseña secreta encontrada es: [escribir_contraseña]</p>

¿Qué entregamos hoy?

Cuando tengan el título, el párrafo, el enlace funcionando y hayan reportado la contraseña secreta, guarden su CodePen y envíen el link URL a la tarea de Google Classroom. ¡Buen trabajo, analistas!
